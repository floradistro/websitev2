================================================================================
                          DESIGN SYSTEM RULES
================================================================================

TYPOGRAPHY:
  ✓ Page titles:    text-2xl md:text-3xl font-semibold
  ✓ Headers:        text-lg font-semibold
  ✓ Labels (caps):  text-[10px] uppercase tracking-[0.15em] font-medium
  ✗ NO font-black, font-light, inline style={{ fontWeight: 900 }}

COLORS:
  ✓ Use white opacity shorthand: white/5, white/10, white/20
  ✓ Backgrounds: bg-black, bg-white/5, bg-white/10, bg-white/15, bg-white/20
  ✗ NO hardcoded hex, NO array notation, NO named colors

BORDERS:
  ✓ border (single pixel) + border-white/10
  ✓ Hover: border-white/20, Active: border-white/30
  ✗ NO border-2, border-4, or thick borders

BUTTONS (4 types):
  ✓ Primary:   bg-white text-black
  ✓ Secondary: bg-white/10 text-white border
  ✓ Tertiary:  text-white/60 border
  ✓ Danger:    bg-red-500/20 text-red-400

SPACING:
  ✓ Padding scale: p-2, p-3, p-4, p-6, p-8
  ✓ Gap scale: gap-2, gap-3, gap-4, gap-6
  ✗ NO random values, NO decimals (p-2.5), NO combinations

MODALS:
  ✓ Overlay: bg-black/80 backdrop-blur-sm
  ✓ Container: bg-black border-white/20 rounded-2xl
  ✗ NO backdrop-blur-xl, NO bg-white/10
